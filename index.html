Three prompts you'll run

Copy these. Paste them into LM Studio with your document attached.

Prompt A — Summarize

[System Context] You are a precise analyst. [Task] Summarize this document in five bullet points. [Constraints] Use only the attached document. [Output Format] Five bullet points. No introduction.

Prompt B — Extract Obligations

[System Context] You are a compliance analyst. [Task] Find every commitment, deadline, or named responsible party. [Constraints] Use only the attached document. Quote each one. [Output Format] Numbered list. Each: Obligation | Owner | Deadline | Quote.

Prompt C — Find The Gap

[System Context] You are a critical reviewer. [Task] What is the single biggest gap this document does not address? [Constraints] Stay within the document. Name the section where it's visible. [Output Format] The gap | Where it appears | Why it matters.
