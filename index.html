The most dangerous assumption

Critical
“When I upload a document, the model reads it.” — It does not. The context window is working memory. The model sees a slice, not the whole document.

What happens between upload and answer

RAG pipeline
Documents are chunked, processed, and stitched back together. For summarization, coverage replaces relevance as the goal.
RAG pipeline: Query → Embed → Vector search → Similar documents → Prompt → LLM → Response. The bottleneck is similarity search, not the model.

You'll leave with three rules

Take-home

These are the rules you'll discover by doing. They work on every AI system — ChatGPT, Copilot, Claude, and anything else.

1
Under 20 pages — attach the file
Model sees most of the content reliably
2
Over 20 pages — paste the section
You control what the model sees. No truncation.
3
Always ask for the exact quote
If it can't quote it, it invented it

These rules do not expire when the model changes.

Where does this break?

Threat model

You're building an agent next year that reads internal policy documents and flags overdue obligations. Where does it fail without anyone noticing?

Truncation
The agent reads 8% of the document. Misses obligations buried on page 40.
Confidence without accuracy
It flags invented obligations with the same tone as real ones.
Document identity confusion
In a batch of ten policies, it attributes an obligation from policy three to policy seven.
No citation
It gives a confident answer but cannot quote the source.

Three prompts you'll run

Copy & paste

Copy these. Paste them into LM Studio with your document attached.

Prompt A — Summarize

[System Context] You are a precise analyst. [Task] Summarize this document in five bullet points. [Constraints] Use only the attached document. [Output Format] Five bullet points. No introduction.

Prompt B — Extract Obligations

[System Context] You are a compliance analyst. [Task] Find every commitment, deadline, or named responsible party. [Constraints] Use only the attached document. Quote each one. [Output Format] Numbered list. Each: Obligation | Owner | Deadline | Quote.

Prompt C — Find The Gap

[System Context] You are a critical reviewer. [Task] What is the single biggest gap this document does not address? [Constraints] Stay within the document. Name the section where it's visible. [Output Format] The gap | Where it appears | Why it matters.
